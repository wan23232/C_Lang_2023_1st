다음과 같은 배열이 있다. 새로운 정수를 읽었을 때에, 오름차순에 맞는 위치에 삽입하는
   함수 insert(int data[], int size, int newitem); 을 작성하라.

** 단, 뒤에서부터 하나씩 테스트하면서 새 데이터가 작으면 기존 데이터를 뒤로 밀고,
   삽입할 자리를 찾으면 그자리에 넣고 종료하는 방식으로 한다. (가장 효율적인 방법이다)

다음과 같은 코드로 테스트한다.

   int score[100] = { 11, 22, 33, 44, 55, 66, 77, 88, 99 };
   int curSize = 9;  // 저장된 항목 수를 나타낸다.
   
   while ( scanf("%d", &item)==1 ){
        insert(score, curSize, item);
	curSize++;
   }

   // 여기서 배열의 모든 항목을 출력.
